⚖️ Delta Formula vs. Proportional Scaling — Why It Matters
// ❌ Proportional model (wrong)
new_$/ft = baseline × (LME_new / LME_base)
// 7/8" example at LME $6.43:
$7.60 × (6.43 / 4.25) = $11.50/ft
// +51 % increase — but mill markup, // Grainger overhead, labor, fittings // do NOT scale with LME. Only raw // copper (~27% of price) changes.

// ✅ Delta formula (correct)
Δ = weight_lb_per_ft × (LME_new − LME_base)
new_$/ft = baseline_$/ft + Δ
// 7/8" L: weight = 0.455 lb/ft
Δ = 0.455 × (6.43 − 4.25) = +$0.99/ft
$7.60 + $0.99 = $8.59/ft ✓
// Market: $7.70 (import) — engine is +11.6% // Much more accurate than +51% overcharge

🔎 2-1/8" Outlier — Root Cause Analysis
❌
2-1/8" pipe is overpriced by +27.9 % vs. the import sale price ($35.82 engine vs. $27.99 market). This is the largest deviation in the dataset. Root cause analysis below.
Distribution Factor Issue

The engine uses dist_factor = 4.303 for 2-1/8" — the highest of any size. This was reverse-engineered from the original Excel table (Grainger at −10%). At current LME, this high factor produces $35.82/ft. The import market provides the same pipe for $27.99 — a factor of just ~3.33×.

Engine dist factor: 4.303×
Implied import factor: 3.33×
Gap: 29 % in distribution factor
Why Grainger Charges More for Large Pipe

Large-diameter copper (2-1/8") is a specialty HVAC item. Domestic distributors (Grainger, Winsupply) charge a premium because: (1) lower turnover, (2) heavier freight costs, (3) domestic mill vs. import, (4) HVAC-grade cleanliness certification. The import site sells uncertified Chinese ACR, which naturally prices lower.

Recommended Action

The engine's 2-1/8" pricing is correct for domestic sourcing. No recalibration needed if the target is Grainger/distribution pricing. However, if the contractor routinely sources large pipe from import suppliers, an optional "sourcing tier" switch (domestic / import) could be added to the UI.

For 75-ton systems using 2-1/8" pipe, the domestic-vs-import price gap translates to $7.83/ft × run length in potential savings — significant on 90 ft runs (~$700 savings on pipe alone).
💡 Recommendations
  • 1
    Keep the delta formula — do not change to proportional scaling The delta formula (new = baseline + weight × ΔLME) is physically correct and matches market data within 4–12 % for mid-range sizes. The proportional model would overcharge by +51 % at current LME. The delta formula is the right choice.
  • 2
    Update insulation costs for large pipe (1-5/8" and 2-1/8") Current: $2.00 and $2.50/ft. Suggested update: $2.75 and $3.50/ft to reflect current ArmaFlex / closed-cell foam market pricing. These values do not change with LME so updates are infrequent (annual review recommended).
  • 3
    Add a "Sourcing Tier" toggle to the UI (Domestic / Import) For large pipe (1-5/8", 2-1/8"), import pricing is 22–28 % below domestic. A toggle would let contractors choose which market they're buying from, applying a simple discount factor (0.78×) to large-pipe sizes when "Import" is selected.
  • 4
    Recalibrate baseline LME annually The current baseline ($4.25/lb) is now 51 % below market. While the delta formula remains accurate for small LME changes, very large LME swings can introduce non-linearity (freight, mill surcharges tier up). Consider resetting the baseline to the current LME once per year by re-pricing the table against current Grainger prices.
  • 5
    Track 3/8" coil vs. stick pricing separately The 3/8" import price ($1.43/ft) is a bulk coil price — roughly 55 % below the engine's stick-based $3.32/ft. This is not an error; it's a purchasing format difference. In the UI, consider noting that 3/8" liquid lines are typically supplied as soft coil by the equipment manufacturer and may not need separate procurement.
